HOW TO BUILD & RUN
------------------

METHOD 1: Manual Build (Recommended for College Projects)
1. Download JavaFX SDK from https://gluonhq.com/products/javafx/
2. Extract to: BakeryManagementSystem/javafx-sdk-21/
3. Run: build.bat
4. Run: run.bat
5. Login: admin / admin123

METHOD 2: Maven Build (Easier - Auto-downloads JavaFX)
1. Install Maven from https://maven.apache.org/download.cgi
2. Run: build-maven.bat
3. Run: run-maven.bat
4. Login: admin / admin123

METHOD 3: Create Windows EXE (For final delivery)
1. Complete Method 1 first (build.bat)
2. Run: create-exe.bat
3. Install package/BakeryManagementSystem-1.0.exe
4. Run like any Windows application
